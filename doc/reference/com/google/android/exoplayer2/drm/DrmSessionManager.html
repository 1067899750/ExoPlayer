      • acquirePlaceholderSession

        @Nullable
default DrmSession<T> acquirePlaceholderSession(Looper playbackLooper,
                                                          int trackType)
        Returns a DrmSession that does not execute key requests, with an incremented reference count. When the caller no longer needs to use the instance, it must call DrmSession.release() to decrement the reference count.

        Placeholder DrmSessions may be used to configure secure decoders for playback of clear content periods. This can reduce the cost of transitioning between clear and encrypted content periods.

        Parameters:
        playbackLooper - The looper associated with the media playback thread.
        trackType - The type of the track to acquire a placeholder session for. Must be one of the C.TRACK_TYPE_* constants.
        Returns:
        The placeholder DRM session, or null if this DRM session manager does not support placeholder sessions.
