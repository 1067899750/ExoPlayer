com.google.android.exoplayer2.text.webvtt

Annotation Type WebvttCue.Builder.TextAlignment


  • @Documented
 @Retention(value=SOURCE)
public static @interface WebvttCue.Builder.TextAlignment
    Valid values for WebvttCue.Builder.setTextAlignment(int).

    We use a custom list (and not Layout.Alignment directly) in order to include both START/LEFT and END/RIGHT. The distinction is important for WebvttCue.Builder.derivePosition(int).

    These correspond to the valid values for the 'align' cue setting in the WebVTT spec.
